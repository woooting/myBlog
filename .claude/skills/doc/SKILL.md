# Doc Reader Instructions

当用户调用此技能或询问相关文档时，请执行以下步骤：

1. **分析用户问题中的关键词**，判断需要参考哪个文档：
   - 功能模块、业务流程、系统概述 → `01-功能模块说明.md`
   - 登录、注册、用户认证、修改密码 → `02-用户认证接口文档.md`
   - 商品、购物车、结算、库存 → `03-商品接口文档.md`
   - 订单、支付、下单 → `04-订单接口文档.md`
   - 订单列表、查询订单 → `05-订单列表接口文档.md`
   - CDK、兑换码、奖品、兑换 → `06-CDK兑换接口文档.md`
   - 充值、钱包、余额、会员 → `07-充值接口文档.md`
   - 数据库、表结构、字段、SQL → `08-数据库表结构.md`
   - 退款、取消订单 → `09-订单退款接口文档.md`
   - 优惠券、领券、折扣 → `10-我的优惠券接口文档.md`

2. **读取文档**
   请使用 file_read 工具读取对应的文档。
   文档基准路径为项目根目录下的: `resource/`
   
   *(注意：建议使用相对路径，Claude 能自动识别项目根目录。如果必须用绝对路径，请确保路径 D:/gitProject/tool/uni-preset-vue-vite/resource/ 是正确的)*

3. **回答要求**
   - 基于文档内容回答用户问题。
   - 必须引用文档中的具体内容（使用 `文件路径:行号` 格式）。
   - 如果问题涉及多个文档，请逐一读取并综合回答。
